# Expo SDK 54

This project targets **Expo SDK 54** (to match the installed Expo Go client).
Read the versioned docs at https://docs.expo.dev/versions/v54.0.0/ before
changing native dependencies, and keep `expo` and all `expo-*`/react-native
packages on their SDK 54 versions (use `npx expo install` / `npx expo install --fix`).
